# Phishing Indicators

Phishing emails try to trick recipients into revealing credentials, sending
money, or installing malware. Common warning signs include:

- Urgent or threatening language ("your account will be suspended", "act now").
- Requests to verify, confirm, or update account details via an embedded link.
- Sender display name that does not match the actual email domain.
- Look-alike domains that swap or add characters (e.g. paypa1.com, mail-paypal.com).
- Generic greetings such as "Dear Customer" instead of your name.
- Unexpected attachments, especially .zip, .html, .iso, or macro-enabled Office files.
- Links whose visible text differs from their real destination.
- Requests for sensitive information: passwords, one-time codes, card numbers.

If several of these indicators appear together, treat the message as high risk.
